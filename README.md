# Snek

Typescript and Node console app rendition of that one snake game.

![Demo](https://imgur.com/0eMqaU0.gif)

### Setup

```bash
$ yarn
$ yarn run dev
$ node .
```

### Play globally

To play globally, run this after building:

```bash
$ yarn global add file:path/to/project
```

Now from anywhere run

```bash
$ snek
```
